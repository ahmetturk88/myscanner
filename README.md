# 🔍 MyScanner

> 🚧 **This project is currently under active development.**

A URL security scanner that detects malware and threats,
powered by the VirusTotal API.

## Tech Stack
- 🐍 Python / Flask
- 🗄️ SQLite
- 🔬 VirusTotal API
- 🎨 HTML / CSS / JavaScript

## Current Features
- ✅ Scan any URL for threats
- ✅ Detailed scan report
- ✅ Previous scans history
